This program converts English sentences to Cued American English using cue notation. Each number and letters corresponds to a handshape and location around the face and chest, as shown in the following diagram:

![alt text](https://github.com/ejt7508/cued-speech/cue-chart.jpg "Cued Speech chart")

Phonemic pronunciation information in International Phonetic Alphabet (IPA) transcription is taken from https://github.com/open-dict-data/ipa-dict


en_US copy.json is the edited en_US.json file which adds oʊɹ to any pronunciation with the ɔɹ sound.
For example, horse originally only had the pronunciation "/ˈhɔɹs/", but was edited to also have the 
pronunciation "/ˈhoʊɹs/". This also includes IPA notation with stress marks ˈ or ˌ in between the characters.
